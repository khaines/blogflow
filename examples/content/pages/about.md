## About BlogFlow

BlogFlow is designed around a few core principles:

- **Git-driven** — content, config, and themes all live in version control
- **Progressive complexity** — start with just Markdown, add config and custom themes as needed
- **Secure by default** — distroless containers, rootless execution, no secrets in images
- **Minimal dependencies** — single static Go binary, no database, no runtime requirements
